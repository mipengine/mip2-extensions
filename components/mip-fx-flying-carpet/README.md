## 说明

视差滚动。

视差滚动组件将其内容包括在一个高度固定的容器中，其表现为容器随着正文上下滚动，而在视觉上，其内容不会随着滚动，从而形成视差滚动。

其内容可以为一张图片，也可以为广告组件。
